CSS3 TIMING FUNCTIONs

Due to the fast development of web technologies, especially CSS3 which carries the timing function that helps HTML elements changing or animating smoothly. I decided to research how timing function is working in web.

explore

What is it?

Timing functions are applied in CSS3 to help HTML elements animation's speed to be faster or slower in a specified duration which the Web Designers can control.

  • Cubic-Bezier()
  • Step()
Why do we use that?
  • CSS2 does not support timing funtions
  • Simpler codes than using Javascript
  • Make HTML elements move & animate smoother and more flexible

cubic-bezier() function
Try out more on http://cubic-bezier.com by Lea Verou
  • Cubic-Bezier() is used to customize the CSS3 timing function manually
  • The timing curve is formed with 4 points: starting (0), point(1), point(2), and ending(3)
  • The starting (0) and ending (3) points are set as default: (0,0) and (1,1)
  • So the cubic-bezier() syntax will only include point 1 & point 2: cubic-bezier( x1,y1, x2,y2 )
  • _x-axis: describes how long an element animates from starting point to ending point which applied on Point 1 before moving to Point 2. Common used : 1 second
  • |y-axis: respresents progression (%) of an element in x time of Point 1 or Point 2

step() function
animation: step infinite 4s steps(4);
  • It is another predefined easing function
  • Not moving smoothly
  • Number of steps basically depend on time (second)
  • Indepent with cubic-bezier()
  • step() syntax: step(n, start/end)
